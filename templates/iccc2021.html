About ICCC2021

ICCC2021 ( Inter College Coding Competition 2021 )

It is an Annual Coding Contest Organised by ICCC team ( now CP Comunity).

It will be a 100 minutes contest.

There will be six problem based on Data Structure and Algorithm.

Students can see past problem on ICCC2020.

It will be hosted on Codechef so it is mandatory to make Codechef ID before registering.

Student can create their Codechef ID at Signup.

You can register yourself for the contest by clicking on the register tab in the navbar.

Why CPC is created ?

  1. To select and prepare team members for the ICPC.
  2. To develop coding environment in our college.
  3. To aware students about coding competitions and coding sites ( Codeforces, Codechef etc. )
  4. To increase coding skills.
  5. To enhance problem solving ability.
  6. To remove code phobia.
  7. To create incubation environment for skilled developer.

About ICCC2020

ICCC2020 ( Inter College Coding Competition 2020 )

It was organised on 28 February 2021 , in IBM Lab.

It will be a 100 minutes contest.

There will be six problem based on Data Structure and Algorithm.

It will be hosted on Codechef so it is mandatory to make Codechef ID before registering.

Student can create their Codechef ID at Signup.

You can register yourself for the contest by clicking on the register tab in the navbar.

Rules and Regulation

  1. CPC is Competitive Programming Community. It is created to help students to get familiar with coding culture and interact with each other.
  2. At CPC, we consider that our environment plays a very important role in our actions, so we decided to create that coding environment in our college.
  3. Our college haven't qualified in ICPC regional ever , this is being our dream to qualify for ICPC regional round. With this aim , we decided to enhance coding culture in our college.
