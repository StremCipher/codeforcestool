ICCC 2020 Problems

Problem 1:

Happy Numbers

Recently, Chef completed a course on Number Theory, now to test his Number Theory Skills. He applied for an online test on Number Theory, He just stuck on a problem and he needs your help. The problem is you are given a number N , you have to tell whether the number is Good or not , a number is said to be Good if square of the number is equal to double of the number.

Input:

First and only line will contain a single integer N.

Output:

If N is a Good Number print "HAPPY" (without quotes), otherwise print "NOT HAPPY" (without quotes).

Constraints :
- 104 ≤ N ≤ 104

Subtask :

100 points : Original Constraints

Sample Input:
9
Sample Output:
NOT HAPPY

EXPLANATION:
9*9=81
9+9=18
81 not equal to 18 so print NOT HAPPY

Problem 2:

Encrypt

A CBI operation is being going against some cyber criminals , so one of CBI agent named Laiba , wants to send some important reports to his senior officer but she decided to encrypt her message so that anyone can't stole his reports, since it contains very confidential information. Message encryption is done such that the order of each character of the string S is reversed. Since she is busy in doing some other work help her encrypt her message.

Input:

The first line of the input contains a single integer T denoting the number of test cases. The description of T test cases follows. The first line of each test case contains a single integer N , the length of the string. The second line contains string S(S is only consist of lowercase English letters).

Output:

For each testcase, output a new string in a new line.

Constraints :
1 ≤ T ≤ 50
1 ≤ N ≤ 106

Subtask :

30 points : 1 ≤ N ≤ 1000
70 points : 1 ≤ N ≤ 106

Sample Input:
1
5
akhil
Sample Output:
zpsro

EXPLANATION:
a=z,k=p,h=s,i=r,l=o.

Problem 3:

Prime Powder

Today Chef buy N magical coke along with N powder from an online sell.Chef arrange all the coke bottle in a queue,each bottle have written a number on it which denotes energy contained in the coke.Number written on them are L,L+1,L+2,â€¦R-2,R-1,R (In left to right order) .Now before drinking the coke he have to dissolve one powder in each coke,the powder have a property that if the energy contained in the coke is only divisible by 1 and itself then new energy of coke is changed to cube of the original energy otherwise it remain same.Chef wants to find the increment in total energy of all coke after dissolving the powder.Since Chef is a busy person so he needs your help.

Input:

The first line of the input contains a single integer T denoting the number of test cases. The description of T test cases follows. The first line of each test case contains two integers L and R.

Output:

For each test case, output energy difference in a single line.

Constraints :
1 ≤ T ≤ 100
2 ≤ L ≤ R ≤ 103

Subtask :

100 points : Original Constraints

Sample Input:
2
4 7
99 100
Sample Output:
456
0

EXPLANATION:
Test case 1: There are 4 cokes of energy 4,5,6,7. Total energy before dissolving the powder = 22. Total energy after dissolving powder = 478. Difference between previous energy to new energy is 456.

Problem 4:

Corona

Recently , novel-coronavirus(nCov) has outbreak in China. It is suppossed to be that this outbreak remains till N days. Scientist are working really hard to find a vaccine for its cure. And they estimated that till (N+1)/2 th day they will able to find its vaccine.
The infection of virus spread till N/2 days and then after N/2 days its effects start decreasing in the same order in which it increases because of the vaccine. And finally at the end of Nth days all cases are either cured or died. For any i from 0 to N−1: The number of people who newly gets infected on ith and (N−1−i)th day are same and the number of people who died on ith and (N−1−i)th day are also same. Since, Chef is working in health department , he wants to know how many people survived on each day. Help him find the number of people who survived on each day from day 0 to N−1.

Input:

The first line of the input contains a single integer Tdenoting the number of test cases. The description of T test cases follows. The first line of each test case contains a single integer N. Nis always odd. The second line contains N space-separated integers A1,A2,...,AN. The third line contains N space-separated integers B1,B2,...,BN.

Output:

For each testcase , print N space-separated integers denoting no. of infected people from day 1 to N.

Constraints :
1 ≤ T ≤ 20
1 ≤ N ≤ 2 * 105
1 ≤ A[i] ≤ 1018
1 ≤ B[i] ≤ 1018

Subtask :

30 points : 1 ≤ N ≤ 103
70 points : Original Constraints

Sample Input:
2
3
6 9 6
2 4 2
5
30 10 20 20 10
17 6 22 17 6
Sample Output:
4 5 4
4 3 8 3 4

EXPLANATION:
Test case 1: Since on the first the number of people who get infected and died will be minimum. Survived on day 1 is 6-2=4 and on day 2 is 5 and then on day 3 as effect deceases due to vaccine so people who survived is 4 .

Problem 5:

Simple Problem 101

Chef created his own function

F(L,R)=f(L) * f(L+1) * f(L+2)....f(R) ;
            Where  f(K)  = 1|2|3|4|5|6...|K;
            for all K greater equal to 0.
            if a|b denotes bitwise OR of a,b;
            Chef needs your help to find F(L,R) for given L and R.
Input:

First line will contain T, number of testcases. Then the testcases follow.
Each testcase contains of a single line of input, two integers L,R.

Output:

For each test case, output is a single line F(L,R) modulo 1000000007.

Constraints :
1 ≤ T ≤ 105
1 ≤ L ≤ R ≤ 109

Subtask :

30 points : 1 ≤ L ≤ R ≤ 103
70 points : Original Constraints.

Sample Input:
2
2 4
5 11
Sample Output:
63
17364375

EXPLANATION:
Test case 1:F(2,4)=f(2)xf(3)xf(4), f(2)=3,f(3)=3,f(4)=7

Problem 6:

Simple Problem 010

Given an array of integers a1,a2,a3...aN of length N.These integers are PIN code of N cities of ChefLand.There is a two way connecting path between each pairs of city ai and aj (1 ≤ i ≤ N ,1 ≤ j ≤ N) if and only if the PIN codes of these pairs of cities are Coprime( ai and aj are Coprime),and length of path would be sum of PIN code(ai + aj).
Chef wants to go to city c from city a via city b,by covering minimum distance.Find the minimum distance chef needs to travel,Or find chef can not reach city c.

Input:

The First line contains integer N, Second line contains N space separated integers a1,a2,a3â€¦.aN ,Third line contains three space separated integers a,b,c.

Output:

Print single integer the minimum distance Chef have to travel,Or -1 if chef can not reach city c

Constraints :
3 ≤ N ≤ 2 * 103
2 ≤ ai ≤ 105 ( all ai are pairwise distinct )
a,b,c must present in given array
Subtask :
10 points : 1 ≤ N ≤ 10
90 points : Original Contraints
Sample Input:
5
2 3 7 9 12

Optimal Path is 2->9->2->3
